# 🚀 Employee Performance Management System - Frontend

A modern React-based frontend application for the **Employee Performance Management System**, designed to provide an intuitive and responsive interface for Administrators, Managers, and Employees.

This application communicates with the Spring Boot REST API backend using JWT Authentication and provides role-based access to different modules of the system.

---

## ✨ Features

### 🔐 Authentication
- Secure Login
- JWT Authentication
- Role-Based Access Control
- Protected Routes

### 👨‍💼 Admin

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Kotteeswaran12/employee-performance-management-ui.git
```

Navigate to the project

```bash
cd employee-performance-management-ui
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
